Configure Jenkins to trigger builds for specific pull requests in Atlassian Stash.

This trigger exposes several environment variables that can be wired into other plugins.

STASH_PULL_REQUEST_ID
The ID of the pull request that needs to be have a preflight build executed.
STASH_PULL_REQUEST_TO_REF
The name of the destination branch.
STASH_PULL_REQUEST_FROM_CHANGESET
The latest commit hash of the source branch.
